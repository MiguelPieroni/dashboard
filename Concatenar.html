Concatenador de SKUs

Procesa y combina códigos SKU de diferentes archivos

1. Códigos de Fabricante e IDSKU

Arrastra tu archivo aquí o haz clic para seleccionar

Archivo cargado:

Requisitos del archivo

  • Formato: Excel (.xlsx, .xls) o CSV
  • Primera columna: Código_de_Fabricante
  • Segunda columna: IDSKU
  • Sin encabezados vacíos o datos faltantes
  • El archivo no puede contener espacios, formulas o formateo especial.
  • No debe contener fórmulas o formateo especial

2. Modelos con generos

Arrastra tu archivo aquí o haz clic para seleccionar

Archivo cargado:

Requisitos del archivo

  • Formato: Excel (.xlsx, .xls) o CSV
  • Primera columna: Modelo
  • Segunda columna: Código de Producto
  • Tercera columna: Género
  • El archivo no puede contener espacios, formulas o formateo especial.
  • Sin encabezados vacíos o datos faltantes
  • Los géneros deben estar escritos de esta forma ( Hombre, Mujer, Unisex, Juvenil, Niño y Bebe )
